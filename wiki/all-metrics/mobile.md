# Mobile metrics

| Domain | Metric name | What the metric measures | User value |
| --- | --- | --- | --- |
| Mobile — voting | Vote sheet opens | Planned multiple-choice voting-sheet opens by bounded voting type and option count. | Reveals whether users reach the choice interaction. |
| Mobile — stories | Story render duration | Planned story payload-to-render duration by bounded layout/media type. | Reveals slow presentation after the backend has already responded. |
