# Posts and media metrics

| Domain | Metric name | What the metric measures | User value |
| --- | --- | --- | --- |
| Posts — voting | Post creation outcomes by voting type | Planned post-creation success/failure split by bounded voting type and option count. | Shows whether publishers can create each supported question format reliably. |
| Media | Media delivery duration | Planned image/video delivery duration by bounded media type and outcome. | Keeps story viewing smooth, especially on slower connections. |
