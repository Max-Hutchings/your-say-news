# Test user feedback — 10 July 2026

## Feed presentation

- Remove the separate headline from each feed post and use the main question as the post's primary heading.

## User characteristics form

- Separate body/health and income/finance questions into distinct pages or sections of the user-characteristics flow.
- Persist answers after every page so that all completed parts of the form are retained if the user leaves and returns later.
- When the user comes back, let them continue from their saved progress without having to re-enter earlier answers.

## Voting and results

- Display the voting data automatically as soon as the user votes instead of showing a separate button that asks them to reveal it.
- Keep the transition from voting to results continuous, without an extra interaction.

## Feed navigation after voting

- While viewing the voting data, an upward scroll or swipe should advance directly to the next post.
